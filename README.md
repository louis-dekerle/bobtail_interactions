Hello dear Traveler!
In this repository you will find the code I used to look a the social network and basic behaviour of a captive population of 8 bobtails in Perth, WA.
The names of the scripts are pretty self-explanatory I hope. 
In the DATA file are the CSV files needed to run the scripts (and their Excel counterparts).
"metadata" and "interaction_matrix" are for the first session of camera trapping while "metadata2" and "interaction_matrix2" are for the second.
The temperature file doesn't appear in the script yet. It is supposed to help me test how much we cannot trust the temperature recorded by camera traps but I haven't had the time to do that yet.
